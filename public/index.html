Oontop-Health API

Welcome to the API documentation page.

The API provides endpoints to manage users and patients in a system. It includes routes for CRUD operations on users and patients, along with middleware for logging, error handling, and CORS support.


Base URL:

http://localhost:8080/api/v1

DB_CONNECTION_STRING:

mongodb://localhost:27017/OonTopApi

Users:

Endpoints to manage users.

  • GET /users : Get all users.
  • POST /users : Create a new user.
  • DELETE /users/:id : Delete user by ID.
  • PUT /users/:id : Update user by ID.
  • POST /users/login : Login user.
  • GET /users/:id : Get user by ID.

Patients:

Endpoints to manage patients.

  • POST /patients: Create a patient.
  • POST /comments: Create a new comment.
  • GET /comments/:id: Get comments by ID patient.
  • GET /patients/:id: Get patient by ID.
  • GET /patients: Get all patients.
  • PUT /patients/:id: Update patient by ID.
  • DELETE /patients/:id: Delete patient by ID.
